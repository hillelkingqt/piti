הוכחת פירוק הטיה-שונות (Bias-Variance Decomposition)

היי עילי, הנה ההוכחה המתוקנת עם כל מה שביקשת! המטרה היא להוכיח ששגיאת הריבועים הממוצעת (MSE) מתפרקת לשלושה רכיבים: הטיה בריבוע, שונות, והשגיאה האי-פריקה.

הגדרות בסיס

נניח מודל סטטיסטי אמיתי: $y = f(x) + \epsilon$, כאשר $\epsilon$ הוא רעש עם תוחלת אפס ($E[\epsilon] = 0$) ושונות $\sigma^2$ ($\text{Var}(\epsilon) = E[\epsilon^2] = \sigma^2$). המודל שאנחנו בונים הוא $\hat{f}(x)$.

שגיאת הריבועים הממוצעת בנקודה $x$ מוגדרת כתוחלת של ריבוע ההפרש בין התצפית $y$ לחיזוי $\hat{f}(x)$:

\text{MSE} = E\left[(y - \hat{f}(x))^2\right]

שלב א׳: פירוק ראשוני

נציב את $y$ בהגדרת ה-MSE:

\text{MSE} = E\left[(f(x) + \epsilon - \hat{f}(x))^2\right]

נסדר מחדש ונפתח את הסוגריים:

= E\left[((f(x) - \hat{f}(x)) + \epsilon)^2\right] = E\left[(f(x) - \hat{f}(x))^2 + 2\epsilon(f(x) - \hat{f}(x)) + \epsilon^2\right]

נשתמש בלינאריות התוחלת:

= E\left[(f(x) - \hat{f}(x))^2\right] + 2E[\epsilon(f(x) - \hat{f}(x))] + E[\epsilon^2]

מכיוון שהרעש $\epsilon$ בלתי תלוי בנתונים ובמודל $\hat{f}(x)$, וגם $E[\epsilon]=0$, איבר התערבות (cross-product term) מתאפס: $2E[\epsilon]E[f(x)-\hat{f}(x)] = 0$. ידוע גם ש-$E[\epsilon^2] = \sigma^2$.

נשארנו עם:

\text{MSE} = E\left[(f(x) - \hat{f}(x))^2\right] + \sigma^2

שלב ב׳: פירוק האיבר המרכזי (הטריק)

כעת נתמקד באיבר $E\left[(f(x) - \hat{f}(x))^2\right]$. נוסיף ונחסיר את $E[\hat{f}(x)]$ בתוך הסוגריים:

E\left[((f(x) - E[\hat{f}(x)]) + (E[\hat{f}(x)] - \hat{f}(x)))^2\right]

נפתח לפי נוסחת כפל מקוצר $(a+b)^2$ כאשר $a = f(x) - E[\hat{f}(x)]$ ו-$b = E[\hat{f}(x)] - \hat{f}(x)$:

= E\left[(f(x) - E[\hat{f}(x)])^2\right] - 2E\left[(f(x) - E[\hat{f}(x)])(\hat{f}(x) - E[\hat{f}(x)])\right] + E\left[(\hat{f}(x) - E[\hat{f}(x)])^2\right]

האיבר הראשון $(f(x) - E[\hat{f}(x)])^2$ הוא קבוע ביחס לתוחלת על $\hat{f}$, וזהו בדיוק ריבוע ההטיה: $\text{Bias}(\hat{f}(x))^2$. האיבר השלישי הוא בדיוק הגדרת השונות של $\hat{f}(x)$: $\text{Var}(\hat{f}(x))$.

נסתכל על איבר התערבות האמצעי: $f(x) - E[\hat{f}(x)]$ הוא קבוע ולכן יוצא מהתוחלת. נשארנו עם $E[\hat{f}(x) - E[\hat{f}(x)]] = E[\hat{f}(x)] - E[\hat{f}(x)] = 0$. לכן, כל איבר התערבות מתאפס.

קיבלנו:

E\left[(f(x) - \hat{f}(x))^2\right] = \text{Bias}(\hat{f}(x))^2 + \text{Var}(\hat{f}(x))

שלב ג׳: הרכבה סופית

נחבר את התוצאות משלב א׳ ו-ב׳:

\text{MSE} = \text{Bias}(\hat{f}(x))^2 + \text{Var}(\hat{f}(x)) + \sigma^2
